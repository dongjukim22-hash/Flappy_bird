# Flappy Bird

A simple Flappy Bird clone made with vanilla JavaScript and HTML5 canvas.

---

### 🛠️ 제작 도구
- Helpy Chat
- GitHubmcp

---

## GitHub Pages 배포 방법

1. **Settings → Pages** 메뉴로 이동합니다.
2. "Source"에서 `main` 브랜치의 `/ (root)`를 선택합니다.
3. 저장 후, 상단에 표시되는 URL로 접속하면 배포된 게임을 볼 수 있습니다.

> 예시: `https://github.com/dongjukim22-hash/Flappy_bird` → `https://dongjukim22-hash.github.io/Flappy_bird/`

---

## 실행 방법
1. 이 저장소를 클론하거나, `index.html`을 브라우저에서 직접 열어 실행할 수 있습니다.
2. 스페이스바 또는 마우스 클릭으로 새를 조작하세요.

본 프로젝트는 Helpy Chat과 GitHubmcp로 자동화되어 제작되었습니다.
